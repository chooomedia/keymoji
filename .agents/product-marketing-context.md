# Product Marketing Context — Keymoji

*Last updated: April 9, 2026*

## Product Overview

**One-liner:** Keymoji is a secure, emoji-based password generator with AI Story Mode and Magic Link / OTP authentication.

**What it does:** Keymoji generates memorable, AI-resistant passwords using emoji sequences in 15+ languages. Users can optionally enable an AI Story Mode (powered by Swiss Apertus AI or their own API key) to create personalised emoji-password stories. Authentication is handled via a 7-digit OTP code sent by email — no passwords needed.

**Product category:** Password manager / password generator (alternative to 1Password, Bitwarden, LastPass for the "memorable password" use case)

**Product type:** SaaS web app (PWA, free + Pro tier)

**Business model:** Freemium — free tier with daily generation limit, Pro tier for unlimited generations + AI Story Mode + advanced security.

---

## Target Audience

**Target users:**
- Privacy-conscious individuals (especially Swiss / DACH region)
- Developers and tech-savvy users who want memorable but secure passwords
- Non-technical users frustrated with complex password requirements
- Enterprises and SMBs that need GDPR-compliant, data-sovereign solutions

**Decision-makers:** Individual users (B2C), IT admins (B2B)

**Primary use case:** Generate a memorable, secure password (emoji sequence) without reusing old passwords.

**Jobs to be done:**
- "Help me create a password I can actually remember"
- "Give me something secure that's not just 'P@ssw0rd1'"
- "Let me use AI to make my passwords more personal and unique"

---

## Problems & Pain Points

**Core problem:** Users forget complex passwords and either reuse simple ones or rely on password managers they don't fully trust.

**Why alternatives fall short:**
- 1Password / Bitwarden: Powerful but complex, require a master password, costly
- Random password generators: Secure but not memorable
- Simple passwords: Memorable but insecure

**What it costs them:** Security breaches, account lockouts, frustration.

**Emotional tension:** "I know I should use better passwords but it's too hard."

---

## Competitive Landscape

**Direct:** Random password generators (Norton, LastPass generator) — no emoji/story mode, not memorable

**Secondary:** Password managers (1Password, Bitwarden) — overkill for many users, not focused on memorability

**Indirect:** Writing passwords on paper, reusing passwords

---

## Differentiation

**Key differentiators:**
- Emoji-based passwords: unique, cross-cultural, memorable
- Swiss AI (Apertus) integration: privacy-first, data stays in Switzerland
- OTP authentication: no passwords, no magic-link clicks
- 15+ languages: truly multilingual
- Free tier available, no credit card required

**Why customers choose us:** Memorable + secure + private + no complex setup.
